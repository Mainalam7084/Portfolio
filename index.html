Sobre mí

Hola, soy un desarrollador web apasionado que disfruta creando sitios rápidos, responsivos y fáciles de usar. Me especializo en desarrollo front-end, y también back-end transformando problemas complejos en código limpio y eficiente. Con una base sólida en HTML, CSS, JavaScript y frameworks modernos, y en back-end JAVA. Mi objetivo es crear experiencias web funcionales y visualmente atractivas.

Tengo 18 años, nací en Bangladesh y actualmente vivo en Madrid, España. Aunque no tengo experiencia profesional, estoy muy motivado para aprender y crecer en el mundo del desarrollo web.

Habilidades técnicas

Desarrollo Front-end

  • HTML5 y CSS3: Crear y maquetar páginas web semánticas, bien estructuradas y adaptables a diferentes dispositivos (responsive design).
  • JavaScript (Básico): Añadir interactividad sencilla a las páginas web, manipular el contenido dinámicamente (DOM) y gestionar eventos.

Desarrollo Back-end (Fundamentos)

  • Java backend nivel medio: Java backend (nivel medio): POO, JDBC, manejo de errores, Git, JUnit, Maven.

Herramientas y Conceptos Clave

  • Conceptos Web Fundamentales: Entender el protocolo HTTP, el modelo cliente-servidor y cómo funciona la comunicación en la web.
  • Herramientas de Desarrollo: Uso eficiente de editores de código (como VS Code), herramientas de depuración del navegador y la lnea de comandos básica.
